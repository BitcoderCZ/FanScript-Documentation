
# TOUCH_STATE
### Type: [float](/MdDocs/Types/Float.md)
Determines which "state" of a touch [touch](/MdDocs/Events/Touch.md) detects.
### Used by

 - [touch](/MdDocs/Events/Touch.md)
### Values
```
TOUCH_STATE_TOUCHING
TOUCH_STATE_BEGINS
TOUCH_STATE_ENDS
```
##### TOUCH_STATE_TOUCHING
Value: 0

When a touch is held for multiple frames.
##### TOUCH_STATE_BEGINS
Value: 1

When a touch starts.
##### TOUCH_STATE_ENDS
Value: 2

When a touch ends.
