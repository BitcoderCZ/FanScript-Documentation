
# TOUCH_FINGER
### Type: [float](/MdDocs/Types/Float.md)
Determines which "finger" (touch) [touch](/MdDocs/Events/Touch.md) detects.
### Used by

 - [touch](/MdDocs/Events/Touch.md)
### Values
```
TOUCH_FINGER_1
TOUCH_FINGER_2
TOUCH_FINGER_3
```
##### TOUCH_FINGER_1
Value: 0

The first touch.
##### TOUCH_FINGER_2
Value: 1

The second touch.
##### TOUCH_FINGER_3
Value: 2

The third touch.
