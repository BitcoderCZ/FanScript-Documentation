# TOUCH_FINGER
### Type: float
Used by [Touch](/MdDocs/Events/Touch.md), determined which finger [Touch](/MdDocs/Events/Touch.md) detects.
```
TOUCH_FINGER_1
TOUCH_FINGER_2
TOUCH_FINGER_3
```
#### TOUCH_FINGER_1
Value: 0

The first finger.
#### TOUCH_FINGER_2
Value: 1

The second finger.
#### TOUCH_FINGER_3
Value: 2

The third finger.
