# Clone(obj, obj)

Creates a copy of [object](#object).

```
void clone(this obj object, out obj copy)
```

## Parameters

#### `object`
Type: obj

The object to copy.

#### `copy`
Type: obj

The copy of [object](#object).

## Remarks

Scripts inside [object](#object) do not get copied inside of [copy](#copy).
