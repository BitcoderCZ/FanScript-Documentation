

# GetScreenSize(float, float)

Gets the size of the screen.

```
void getScreenSize(out float width, out float height)
```

## Parameters

#### `width`
Type: [float](/MdDocs/Types/Float.md)

Width of the screen (in pixels).

#### `height`
Type: [float](/MdDocs/Types/Float.md)

Height of the screen (in pixels).
