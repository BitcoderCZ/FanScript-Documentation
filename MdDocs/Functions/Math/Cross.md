# Cross(vec3, vec3)

Returns the [cross product](https://en.wikipedia.org/wiki/Cross_product) of [vector1](#vector1) and [vector2](#vector2).

```
vec3 cross(vec3 vector1, vec3 vector2)
```

## Parameters

#### `vector1`
Type: vec3

The first vector.

#### `vector2`
Type: vec3

The second vector.

## Returns

vec3

Cross product of [vector1](#vector1) and [vector2](#vector2).
