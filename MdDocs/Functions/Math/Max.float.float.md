

# Max(float, float)

Returns the larger of 2 numbers.

```
float max(float num1, float num2)
```

## Parameters

#### `num1`
Type: [float](/MdDocs/Types/Float.md)

#### `num2`
Type: [float](/MdDocs/Types/Float.md)

## Returns

[float](/MdDocs/Types/Float.md)

Returns either [num1](#num1) or [num2](#num2), depending on which one is larger.
