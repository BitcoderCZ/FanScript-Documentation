# Dist(vec3, vec3)

Returns the distance between [vector1](#vector1) and [vector2](#vector2).

```
float dist(vec3 vector1, vec3 vector2)
```

## Parameters

#### `vector1`
Type: vec3

The first vector.

#### `vector2`
Type: vec3

The second vector.

## Returns

float

The distance between [vector1](#vector1) and [vector2](#vector2).
