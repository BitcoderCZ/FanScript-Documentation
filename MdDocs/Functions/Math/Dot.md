# Dot(vec3, vec3)

Returns the [dot product](https://en.wikipedia.org/wiki/Dot_product) of [vector1](#vector1) and [vector2](#vector2).

```
float dot(vec3 vector1, vec3 vector2)
```

## Parameters

#### `vector1`
Type: vec3

The first vector.

#### `vector2`
Type: vec3

The second vector.

## Returns

float

Dot product of [vector1](#vector1) and [vector2](#vector2).
