# LinearLimits(constr, vec3, vec3)

Sets the linear limits of [constraint](#constraint).

```
void linearLimits(this constr constraint, vec3 lower, vec3 upper)
```

## Parameters

#### `constraint`
Type: constr

The constraint to apply the limits to.

#### `lower`
Type: vec3

The lower limit.

#### `upper`
Type: vec3

The upper limit.
