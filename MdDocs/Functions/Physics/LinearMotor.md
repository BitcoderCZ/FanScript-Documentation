Makes [constraint](#constraint) move.


## Parameters

#### `constraint`
Type: [constr](/MdDocs/Types/Constr.md)

Type: [vec3](/MdDocs/Types/Vec3.md)

The speed at which to move at.

#### `force`
Type: [vec3](/MdDocs/Types/Vec3.md)
