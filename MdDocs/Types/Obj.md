
# Obj

A fancade object.

```
obj
```

## How to create

``` fcs
obj a = getObject(5, 0, 0) // gets the object located at (5, 0, 0)
obj b = getBlockById(BLOCK_GRASS) // places the GRASS block (id 3) and returns a reference to it
```
